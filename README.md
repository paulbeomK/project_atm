# Project_atm
-------------------------------------------
A simple ATM controller implemented in C++.


### Description
-------------------------------------------
Main application (main.cpp)
- Demo through a consol-based simulation.
- Feature
    - Card Insertion & Ejection
    - PIN Authentication
    - Balance Inquiry
    - Deposit
    - Withdrawal
    - Multiple Accounts per card

### Requirements
-------------------------------------------
- C++17 or later
- CMake 3.10 or later
- C++ compiler

### Build & Test
-------------------------------------------
`cmake CMakeLists.txt`
`make`

- Main
    - `./project_atm`
- unit_test
    - `./test_atm`
